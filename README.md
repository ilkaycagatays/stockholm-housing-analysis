# 🏠 Stockholm Housing Price Analysis

## 📌 Overview
This project analyzes the Stockholm housing market using real data 
from SCB (Statistics Sweden) API. We examine price index trends 
between 2015 and 2024.

## 🔍 Key Finding
Stockholm housing prices peaked in 2022, followed by a decline 
driven by Riksbank interest rate hikes.

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- SCB PxWeb API (real government data)

## 📊 Data Source
Statistics Sweden (SCB) — Official Swedish government statistics
https://www.scb.se
